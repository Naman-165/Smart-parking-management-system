# Smart-parking-management-system
The Smart Parking Management System is a web-based application developed to manage parking spaces efficiently using a digital platform. It provides real-time information about available, occupied, and reserved parking slots through an interactive dashboard.

The system allows the admin to assign parking slots to vehicles, search vehicle details, and monitor parking usage across multiple blocks. It helps reduce manual effort, saves time, and improves parking space utilization.

🎯 Key Features

✨ Real-time parking slot status
✨ Interactive parking map (ET, MT, HM blocks)
✨ Admin dashboard with live statistics
✨ Auto parking slot allocation
✨ Vehicle search using registration number
✨ Reservation system
✨ Parking fees tracking
